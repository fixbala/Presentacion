# **App Name**: Portfolio Revamp

## Core Features:

- Project Showcase: Display a visually appealing grid of my projects with descriptions and links, reflecting the style from fixbala.github.io/Portafolio/
- Skills Visualization: Interactive display of my skills and technologies proficiency.
- About Me Section: Dynamic presentation of my professional background, experience, and objectives.
- Contact Form: Modern contact form integrated for easy communication, supporting automated feedback. 
- AI-Powered Recommendation: Use an AI tool that recommends which projects from the portfolio are best suited to the viewer, based on user description or other viewer parameters. The LLM tool will reason to choose the top recommendations.

## Style Guidelines:

- Primary color: Deep blue (#293462), conveying trust and stability in a modern way.
- Background color: Very light blue (#E0F4FF), providing a soft, unobtrusive backdrop that complements the primary color without overpowering it.
- Accent color: Vibrant magenta (#E4007C), drawing attention to interactive elements and injecting a sense of energy and creativity.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Custom designed, minimalist icons related to skills and project categories, using the magenta accent color for highlights.
- Responsive grid layout that adapts smoothly to different screen sizes and devices, with projects displayed prominently.
- Subtle hover effects and transitions to enhance user engagement and provide visual feedback on interactions, such as project selection or form submission.
